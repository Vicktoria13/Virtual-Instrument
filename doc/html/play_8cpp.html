Detailed Description

fichier cpp contenant les fonctions appelées dans le programme principal. Permet de jouer des instruments virtuellement via un objet tracké via sa couleur.

Author
Victoria Nguyen et Matthieu Sysouvanh ROB3
Version
0.1
Date
2022-06-08

Fichier regroupant toutes les fonctions nécessaires pour chaque mode. ATTENTION : IMPORTANT :

  • Il faut absolument modifier la LIGNE 91 avec le chemin absolue depuis la racine qui mene au dossier sampples piano situé dans projetLibre
  • Si le chemin d'ecriture lors de l'ouverture du fichier port série ne fonctionne pas, il y aura une erreur : le chemin actuel est /dev/ttyACM0. Changer la variable path pour que le port ou est branché la carte soit le bon !

LIGNE 91

*Si les servos ne sont pas branchés/ relié a l'ordinateur, le programme ne fonctionne pas
